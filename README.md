# monthly-dinner

App mobile-first para coordinar cenas mensuales grupales.

## Stack

- Next.js 14 App Router
- TypeScript estricto
- Supabase Auth + Postgres + RLS
- Design system basado en Tailwind + primitives estilo shadcn/ui

## Setup local

1. Clona el repositorio y haz checkout a `fresh-start`.
2. Copia `.env.local.example` a `.env.local` y completa las credenciales de Supabase.
3. Ejecuta `npm install`.
4. Ejecuta `npm run dev`.
